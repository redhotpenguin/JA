== Description ==

This plugin is designed to add a link to users.php to allow admins to edit user's BuddyPress Profiles.

== Installation ==

Before you start you should have BuddyPress installed and activated.

= 1. Upload to your plugins folder, usually `wp-content/plugins/` =

= 2. Activate the plugin on the plugin screen. =

= 3. On the Users page, when you hover over a user's name you will now see a "Edit BuddyPress Profile" link. =
